Fulfillment setup path

  1. Sync the published Printful products into /merch/printful-storefront-catalog.json.
  2. Deploy the SmartSleeve Merch Checkout Worker with Stripe test keys and Printful draft-order fulfillment.
  3. Set this page's merch checkout endpoint to the Worker /checkout route.
  4. Complete one Stripe test checkout and review the draft order in Printful before turning on live order confirmation.

Pricing note

Prices shown here are generated from the published Printful product variants, not typed into the website by hand. If you adjust a product price in Printful, rerun the catalog sync before publishing the site.

Current checkout buttons are merchant-of-record ready but disabled until smartsleeve-merch-checkout-endpoint is configured. Card data stays in Stripe-hosted checkout, never on this static site.
