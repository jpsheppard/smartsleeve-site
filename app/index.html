Fulfillment setup path

  1. Create a Stripe account under SmartSleeve Quantitative Trading Systems, LLC and connect a SmartSleeve business bank account.
  2. Create the tee products in a print-on-demand vendor account owned or paid by SmartSleeve, using the print PNGs above.
  3. Deploy the SmartSleeve Merch Checkout Worker and configure this page's merch checkout endpoint.
  4. Configure the Stripe payment-success webhook to submit paid orders to the fulfillment vendor and store Stripe/vendor order ids.

Quality note

For lowest friction, launch with direct-to-garment black tees. For a premium drop, use embroidered caps or embroidered chest marks where the provider supports it; large full-color art is usually better printed than embroidered.

Current checkout buttons are merchant-of-record ready but disabled until smartsleeve-merch-checkout-endpoint is configured. Card data must stay in Stripe-hosted checkout, never on this static site.
